1. Deutsche Tastatur auf dem Tablet Einstellen
    Aplications ---> Settings --->Keyboard ---> Layout ---> German (English Löschen)

Tablet IP 192.168.100.160
Auf dem Router einstellen 

2. Passwort aud dem Tablet ändern: 
    Konsole: passwd

WICHTIG!!!
    sudo apt update
    sudo apt upgrade

3. Installer und Ordner auf Tablet Kopieren
    Install Scripts auszuführen
        auto_installer.sh

        Datei erst ausführbar machen
            chmod +x auto_installer.sh
        Datei auszuführen
            ./auto_installer.sh


3. User Anlegen
    5.1 User owipex_usr hinzufügen
        adduser owipex_usr
    5.2 User in Sudoer List hinzufügen
        usermod -aG sudo owipex_usr
    
6. sudo installieren
    6.1 apt install sudo

7. ONBOARD Tastatur installieren
    apt install onboard

8. XHOST Freigabe für benutzer
    8.1 Bevor du den xhost Befehl ausführst, musst du die DISPLAY Umgebungsvariable für den root Benutzer setzen. Du kannst dies tun, indem du den folgenden Befehl ausführst:   
        export DISPLAY=:0
    8.2 Danach solltest du in der Lage sein, den xhost Befehl erfolgreich auszuführen:
            xhost +SI:localuser:owipex_usr
